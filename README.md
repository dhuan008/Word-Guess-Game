## Game Objective:
```
    --- Guess the correct movie to gain a point
```

#### Game Instructions:
```
    1. Press any letter to start
    2. Keep on guessing to complete the word
    3. If you get the word right, you get a point
    4. If you run out of guesses, you lose the game
```
